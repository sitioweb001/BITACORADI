Coloca los archivos firmatecnico.png (o .jpg) y sello.png (o .jpg) en la misma carpeta que index.html para que aparezcan automáticamente en cada actividad y en el documento descargado. Si no existen, puedes subirlos aquí (solo se guardan en este navegador).

⚠️ El navegador no dejó leer firmatecnico/sello desde la carpeta local para generar el PDF (pasa cuando abres index.html con doble clic en vez de subirlo a un hosting). La firma/sello sí se ven en pantalla, pero el botón "Descargar PDF" puede fallar. Para evitarlo, usa los botones "Subir firma" / "Subir sello" de abajo (seleccionando el mismo archivo), o publica esta carpeta en un hosting (Netlify, GitHub Pages, etc.) y ábrela desde ahí.
